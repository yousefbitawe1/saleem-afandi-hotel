# 🏨 Saleem Afandi Hotel — فندق سليم أفندي

> A modern, bilingual (Arabic / English) hotel website built with pure HTML, CSS, and JavaScript — no frameworks, no dependencies.

🌐 **Live Demo:** [View Website](https://yousefbitawe1.github.io/saleem-afandi-hotel/)

---

## ✨ Features

- 🌍 **Bilingual** — Full Arabic (RTL) & English (LTR) support with instant switching
- 🌙 **Dark / Light Mode** — Toggle with one click, preference saved in localStorage
- 🖼️ **Cinematic Room Gallery** — Film-strip thumbnails, fade transitions, dot navigation
- 📱 **Fully Responsive** — Mobile-first design, touch swipe on gallery & lightbox
- 💬 **Floating WhatsApp Button** — Appears on scroll for instant contact
- ⬆️ **Back to Top Button** — Smooth scroll back
- 📊 **Reading Progress Bar** — Gold gradient bar at the top
- ⭐ **Testimonials Carousel** — Auto-rotating with manual dot control
- 🗺️ **Embedded Google Maps** — Interactive live map in the contact section
- 🏠 **Check-in / Check-out Times** — Clearly displayed time cards
- 🔗 **Social Links** — WhatsApp, Instagram, TikTok, Facebook, Google Maps, Waze
- ⚡ **Lazy Loading** — All images load only when needed
- 🎨 **Scroll Reveal Animations** — Elements animate in as you scroll

## 🚀 Getting Started

No build tools or installations needed. Just open the file:

```bash

git clone https://github.com/your-username/saleem-afandi-hotel.git

# Open in browser
open index.html
```

Or simply download the ZIP and open `index.html` directly.
